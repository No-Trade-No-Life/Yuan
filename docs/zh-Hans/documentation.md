# 文档

[@yuants/docs](ui/docs) 是 Yuan 的文档。

它由 [Docusaurus](https://docusaurus.io/) 构建。您可以在 [这里](https://www.ntnl.io/) 找到最新的文档。

## 文档架构

### 技术栈

- **Docusaurus**: 现代化的文档网站生成器
- **Markdown**: 简洁的文档编写格式
- **React**: 交互式组件支持
- **TypeScript**: 类型安全的代码示例

### 文档特点

- **多语言支持**: 支持中文和英文文档
- **版本管理**: 支持多版本文档管理
- **搜索功能**: 全文搜索支持
- **响应式设计**: 适配各种设备屏幕
